# ipod-portfolio

An interactive iPod-style portfolio website.

## Features
- **Click Wheel Navigation**: Authentic iPod Classic experience.
- **AI-Powered**: Showcases AI projects and expertise.
- **Music & Games**: Integrated music player and classic games.
- **Tech Stack**: Next.js 14, TypeScript, Framer Motion, and Styled Components.

## Quick Start
```bash
npm install
npm run dev
```

Visit [yashvoladoddi.vercel.app](https://yashvoladoddi.vercel.app) to check it out.
